Detailed Description

This class handles all movement related behaviors in 3D The host object (mentioned in blow) is the object which this script will perform its functions on .

!– The host object is not necessary be the parent gameobject of the script.!–

This function received the player's inputs (forward and side) then saved them to be processed in the next fixed update.

If shouldMoveTowardCameraDirection is true then it will rotate the host game object toward the camera's faciing direction first.

Parameters
forwardfordward is how much the host game object should move forward and backward
sideside is how much the host game object should move sideway

void Movement.Step ( float  speed,
int  forward,
int  side 
)
private

This function move the host object according to the forward and side inputs of the player.

The movement is performed by manipulating the velocity property of the rigidbody. The speed of the movement is reduced if the host object is airborned.

Parameters
speedspeed is the desired movement speed for the host object.
forwardforward is how much the player need to move forward or backward.
sideside is how much the player need to move sideway.
